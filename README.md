# 3DBPP-Solver-Server

## How to install

1. In a terminal, create a new virtual environment
```
virtualenv my_env
```

2. Activate it
```
my_env\Scripts\activate.bat
```

or (on Linux / Mac)
```
source my_env/Scripts/activate
```

3. Install the requirements
```
pip install -r requirements.txt
```

4. Done! Enjoy your server
